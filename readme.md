# Budgets - Node JS Application

# Development Setup
Install requirements

`npm install`

Fill .env file with port number, debug, etc

Install needed global packages using NPM:

`npm install -g typescript`

`npm install -g grunt`

Start dev server using:

`npm run dev`

# Models

Prisma is used to migrate models into the DB.

Install

`npm install -g prisma`

And migrate

`prisma migrate dev`
